IoTSharp Edge Runtime

IoTEdge

C# AOT 边缘网关基座。它在现场侧读取本地 bootstrap，注册到 IoTSharp，拉取采集配置，执行工业协议轮询，并把结果上传到 IoTSharp、SonnetDB、InfluxDB 或 ThingsBoard。

快速开始 查看源码
Bootstrap 本地配置 bootstrap.json / local-collection.json
Collect 协议采集 Modbus / PLC / OPC UA / MTConnect
Upload 多目标上报 IoTSharp / SonnetDB / InfluxDB

Capabilities

边缘节点需要的运行能力，集中在一个宿主里

01

平台同步

向 IoTSharp 注册、心跳、上报能力，并周期性拉取采集配置。

02

离线运行

平台不可用时继续使用本地缓存配置运行，不清空已有执行计划。

03

工业驱动

已接入 Modbus、Siemens、Mitsubishi、Omron FINS、Allen-Bradley、OPC UA、MTConnect。

04

BASIC 脚本

内置 BASIC runtime，并支持 MQTT、串口、Modbus、PLC 等采集内建能力。

05

本地管理 UI

随宿主启动的 Vue SPA 提供运行态、配置、日志、协议和上传链路编辑入口。

06

AOT 发布

支持普通构建和 AOT 发布，AOT 模式自动跳过可选网关扩展注入层。

Architecture

从平台模型到本地执行缓存

IoTSharp 平台采集模型、节点管理、任务下发

Industrial Devices协议驱动、点位读取、写入控制

IoTEdge.Application 运行服务、配置快照和转换服务。

IoTEdge.Infrastructure SQLite 缓存、驱动适配和上传通道。

IoTEdge 单宿主程序、本地 API、同步 worker 和 SPA 托管。
